# Nexus AI Customer Support System

Multi-agent AI customer support with TypeScript, React, and RAG. Routes queries to specialized agents for intelligent responses.



- **4 Specialized Agents**: Router, Knowledge, Action, Escalation
- **RAG System**: Vector search with Pinecone


## Tech Stack

**Backend**: TypeScript, Node.js, Express, PostgreSQL  
**AI**: AWS Bedrock Nova  
**Vector DB**: Pinecone  
**Frontend**: React, TypeScript

## Agents Explained

- **Router**: Analyzes intent, routes to correct agent
- **Knowledge**: RAG-based answers from docs
- **Action**: Executes tasks (orders, accounts)
- **Escalation**: Handles complaints, creates tickets
